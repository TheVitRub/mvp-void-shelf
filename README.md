# MVP - Минимально жизнеспособный продукт

Модуль MVP содержит основную функциональность системы мониторинга наполнения полок.

## Структура модуля

- **main.py** - Главный скрипт для запуска системы мониторинга
- **config.py** - Конфигурационные параметры системы
- **camera/** - Модуль для работы с IP-камерами через RTSP
- **area_calculation/** - Модуль расчета площади объектов и процента наполнения
- **show_picture/** - Модуль визуализации результатов детекции
- **track/** - Модуль трекинга объектов с использованием ByteTrack
- **outline_the_shelves/** - Инструмент для калибровки координат полок

## Быстрый старт

1. Настройте переменные окружения в `.env` файле
2. Укажите путь к модели YOLO в `config.py`
3. Подготовьте JSON файл с координатами полок (используйте `outline_the_shelves/calibrate_shelf_coordinates.py`)
4. Запустите: `python MVP/main.py`

## Калибровка координат полок

Используйте инструмент `outline_the_shelves/calibrate_shelf_coordinates.py` для интерактивного определения координат полок на изображении.

Подробная документация находится в комментариях каждого модуля.
